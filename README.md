# ParalelOrnekleriPython

Python ile geliştirilmiş **paralel işlem örnekleri** içerir. 
Bu proje, işlem sürelerini azaltmak ve CPU kullanımını optimize etmek için hazırlanmıştır. 
Özellikle çok çekirdekli sistemlerde paralel programlamayı öğrenmek isteyenler için uygundur.

## Özellikler
- Paralel döngü ve işlem örnekleri
- İşlem sürelerini ölçme ve karşılaştırma
- Basit ve anlaşılır Python kod yapısı

## Kurulum
1. Projeyi klonlayın: `git clone <repo_link>`
2. Python 3 yüklü olmalı
3. Terminal üzerinden çalıştırın: `python main.py`
